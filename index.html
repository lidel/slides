IPFS COMPANION

IPFS Companion

browser extension

Bold mission..

Add IPFS support
to web browsers

Key features

  • uplifts IPFS paths on the web
  • quick file sharing
  • opt-in experiments: embedded js-ipfs, dnslink
  • IPFS API under window.ipfs

IPFS addressing

& protocol uplift

↓ http://foo.tld/ipfs/Qm..MnR/cat.jpg
→ [local gateway]/ipfs/Qm..MnR/cat.jpg

  • detect & validate requests for IPFS paths
    • including cross-origin XHRs
  • redirect & serve from IPFS node
  • start using content-addressing today!

window.ipfs

  • safe subset of IPFS API
  • conservative ACL controls
  • MFS scoped per app
  • a living prototype, feedback is welcome!

What's next for IPFS in Web Browser?

Tackling Open Problems

  • Transition to Base32 CIDv1 (ipfs://)
  • UX: ship new WebUI with Companion
  • File upload: browser bugs and performance
  • Embedded js-ipfs with DHT and Circuit Relay
  • WebRTC transport in Web Workers
  • IPFS Gateways in Service Workers (P2P CDN!)
  • New APIs: native protocol handlers, DNS TXT Lookup, mDNS, TCP/UDP Sockets, FS access
